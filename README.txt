Life, the Universe, and Everything
----------------------------------

This package replace the standard Plone edit view tabs for a collapsible UI.

Don't Panic
-----------

Configure
^^^^^^^^^

- Go to 'Site Setup'.
- Select 'sc.collapsible.edit'.
- Enable.
